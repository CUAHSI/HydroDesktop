1) You need to get the DotSpatial dlls from http://dotspatial.codeplex.com/releases/view/latest
2) You need to set the reference path to those dlls.
	a) Select the project in Solution Explorer and choose Properties
	b) Go to the Reference Paths tab
	c) Browse to the location of the DotSpatial dlls
	d) Click Add Folder
3) You need to make sure you are using .Net 4
4) You may wish to setup Debugging to launch MapWindow and set your build path to the plugin folder of MapWindow.
