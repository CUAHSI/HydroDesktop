To run the model.

1. Double click start_muskingum_service.bat to start the Muskingum service.  This command screen will remain open while to model runs.  
[This assumes you have Python and the necessary Python libraries installed.]  

2. Double click run-SmithBranchModel-01.bat to run model. Results from model are written to the root directory.

3. Close service command line window to end Muskingum service.
