EPA ToolÂ¶

The EPA Tool in HydroDesktop provides access to EPA’s WATERS Web and Database Services. Users can use this tool to get the watershed and the flowlines that are delineated from the specified point on the map. Both the watershed and flowlines are based on NHD (National Hydrography Dataset) system.
