EPA ToolsÂ¶

The EPA Tools in HydroDesktop provide access to EPA’s WATERS Web and Database Services. Users can use these tools to get the watershed and the flowlines that are delineated from the specified point on the map. Both the watershed and flowlines are based on NHD (National Hydrography Dataset) system.
