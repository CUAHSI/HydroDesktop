IntroductionÂ¶

The Graph View is a HydroDesktop extension that provides functionality for visualizing data that have been downloaded and stored in the HydroDesktop data repository database. The Graph View enables you to generate a variety of plot types for time series data as well as simple summary statistics that can be used for exploratory data analysis. A variety of plot customization options are available for changing the look and feel of the plots generated by the Graph View extension.
