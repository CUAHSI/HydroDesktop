*.omi file
==========

*OpenMI model file*.  An OpenMI model file (or *.omi file) is a single component that can be used to create a component composition.  Components can be models or tools for working with data. Each component has input and output exchange items that allow the component to be linked to other components. Below is an image of what a .omi file when loaded into HydroModeler.
