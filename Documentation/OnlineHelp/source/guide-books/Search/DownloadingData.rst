Downloading Data
================

**To download data:**

1. In the **Search Panel**, in the **Results** tab, type a theme name in the **New Theme** text box. This will be the name of the downloaded data map layer and time series group.
2. Click **Download Data**.
3. A download manager window opens to show the progress of download.

.. figure:: ./images/Search_fig07.png
   :align: center
.
   
Once the download is complete, the new theme is shown under the Themes group in the Map Contents. If you create additional themes, they will also appear in the Themes group.
