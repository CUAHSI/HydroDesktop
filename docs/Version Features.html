Roadmap

Version 1.6 will include improved/updated existing functionality. Goals for version 1.6 include:

  1. Refactor the Search Keywords to be simpler and more intuitive as per http://hydrodesktop.codeplex.com/workitem/8569
  2. Refactor the Search Data Sources to show more info about sources, sort, etc. as per http://hydrodesktop.codeplex.com/workitem/8500. This needs some additional discussion and resolution on what attributes to make available and whether to use a tree or table, and whether to use space or time or something else as main organizing feature.
  3. Make online plugins updatable and possible autoupdated
  4. Additional bug fixes as they are found. 

Version 2.0 will be our first "HydroShare" version of HydroDesktop. Goals for version 2.0 include: 

  1. Support for bringing in a shapefile of sites and allowing it to be treated as a search results layer - also should include online OGC WFS web services.
  2. Uploading and sharing data packages
